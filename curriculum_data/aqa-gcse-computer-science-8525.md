# AQA GCSE Computer Science (8525) - Mastery Map

## CORE FOCUS
This syllabus focuses on computational thinking, understanding how computers work, and developing programming skills (specifically Python, C#, or VB.NET).

## PAPER 1: Computational Thinking and Programming Skills
- **Section A: Computational thinking**
  - Understand algorithms (abstraction, decomposition, algorithmic thinking).
  - Interpret, correct, and complete algorithms.
  - Understand standard algorithms (linear search, binary search, bubble sort, merge sort).
- **Section B: Programming skills**
  - Data types (integer, real, boolean, character, string).
  - Programming concepts (variable declaration, constant declaration, assignment, iteration, selection, subroutine).
  - Arithmetic operations, relational operations, Boolean operations (AND, OR, NOT).
  - Data structures (arrays, records).
  - File handling (reading to/writing from text files).

## PAPER 2: Computing Concepts
- **3.3 Fundamentals of data representation**
  - Number bases (decimal, binary, hexadecimal).
  - Converting between bases, binary arithmetic, overflow.
  - Character encoding (ASCII, Unicode).
  - Representing images and sound (pixels, resolution, color depth, sample rate).
  - Data compression (Huffman coding, RLE).
- **3.4 Computer systems**
  - Hardware and software.
  - Boolean logic (AND, OR, NOT, XOR truth tables and logic circuits).
  - Software classification (system software, application software, OS functions).
  - Systems architecture (Von Neumann architecture, CPU components: ALU, CU, Registers, Clock).
  - Memory and Storage (RAM, ROM, Cache, Magnetic, Optical, Solid State).
- **3.5 Fundamentals of computer networks**
  - Types of networks (LAN, WAN).
  - Network topologies (Star, Bus).
  - Network protocols (TCP/IP stack: Application, Transport, Internet, Link layers).
- **3.6 Cyber security**
  - Cyber security threats (social engineering, malware).
  - Methods to detect and prevent vulnerabilities (passwords, biometrics, CAPTCHA, email confirmation, firewalls, MAC address filtering, encryption).
- **3.7 Relational databases and SQL**
  - SQL commands (SELECT, FROM, WHERE, ORDER BY, INSERT, UPDATE, DELETE).
- **3.8 Ethical, legal and environmental impacts**
  - Impact of digital technology on wider society, ethical issues, legislation (Data Protection Act, Computer Misuse Act, Copyright Designs and Patents Act).

## ASSESSMENT OBJECTIVES (AOs)
- **AO1:** Demonstrate knowledge and understanding of the key concepts and principles of computer science.
- **AO2:** Apply knowledge and understanding of key concepts and principles of computer science.
- **AO3:** Analyse problems in computational terms: to make reasoned judgements and to design, program, evaluate and refine solutions.
