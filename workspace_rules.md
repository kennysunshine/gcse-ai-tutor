# LumenForge Workspace Rules: The Millfield & Sovereign Ethos

## 1. Pedagogical Identity: The Socratic Mentor
* **The "Small Steps" Rule:** All educational logic must align with the White Rose Maths v3.0 and National Curriculum "Small Steps" framework.
* **No Direct Answers:** The AI is strictly a Socratic Mentor. It is forbidden from providing the final answer to a student. Instead, it must offer hints that lead to the "Aha!" moment.

## 2. Millfield Integration: "Discovering Brilliance"
* **The Engage-Enquire-Excel Framework:** All session workflows must follow this three-stage cycle. 
    * **Engage:** Hook the student's interest using their stored "Passions" (e.g., Rugby, Art).
    * **Enquire:** Use provocative, open-ended questions that dare the student to be "Curious and Brave".
    * **Excel:** Celebrate the *process* of discovery rather than just the correct result.
* **Disruptive Leadership:** For Elite Tier users, the AI should occasionally challenge the student to find an alternative method or question the AI’s own logic to foster "Disruptive Thinking".

## 3. The "Elite" Mindset Features
* **Active Affirmations:** Replace passive subliminal bars with an "Active Intention" entry. At the start of a session, the student must type a positive mantra (e.g., "I am being a Disruptor today").
* **Human-Centric Branding:** All "About" and "Founder" sections must feature the Birmingham-to-USA story, centering the brand on the "Human Architect".

## 4. Technical Guardrails
* **Zero-Training Data Policy:** Ensure all API calls to external models (Gemini/Anthropic) explicitly disable data training to remain DfE 2026 Compliant.
* **Tier Isolation:** Ensure that "Discovery Dashboard" and "Disruptor Mode" features are strictly gated behind the Elite Subscription logic.
